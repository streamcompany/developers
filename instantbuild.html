Pre-requirements for building.

50mbps Internet Connection
1 Server
1 Web Beacon
1 Android and iOS test device
1 Windows PC

  1. BUILD - Get ready to build your Instant Search Engine. To start, make sure you have Android device near you with the Instant App Installed. On the latest version, make sure to clear all app data. Open up Instant, then tap the "+" when it asks for your search engine. A dialogue will appear with a text entry to enter your URL. Type in a URL that your server is pointed to. On your Windows PC, grab a copy of your search engine, however allow it to just be the <input> box itself. Plant this and link it back to your original URL, replacing all "/..." with "https://example.com/...". This makes sure that your server is connected to your search result page. Now, with your search engine, make a *.html page and name it input.html. Insert your search box into a <div> in the <body>, make sure the margins are responsive rather than set to a figure as this could cause issues for displays. (Optional: You may add in anything under the Search Bar e.g Quick Links, News, Services. However it may affect performance.) Now, also in the top directory create a file named inshd.html. Here, this is your header on the app, you can put whatever artwork, colours and text you need to put, however, bear in mind that it will appear differently on different screen sizes, so keeping responsive and in the middle would be reasonable. Finally, you need to close the Apache and the MySQL ports to allow faster connections in and out. We can now move onto security.
  2. SECURITY - This bit is a simple, if you are a personal or small company then this step is optional. However, if you aren't, please get with DuckDuckGo for the security programs.
